# Gas Leak Detection System

## Introduction
An IoT-based gas leak detection system designed to monitor gas concentration and provide alerts when gas levels exceed a safe threshold.

## Technologies
- Arduino Uno
- MQ Gas Sensor
- C++
- IoT Fundamentals

## Features
- Gas concentration monitoring
- Real-time detection
- LED warning
- Buzzer alarm
